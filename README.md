# SnapShare
SnapShare is a full-stack social media web application built with Django, replicating core features of modern social networking platforms. Developed as a BIM final year project at Hetauda School of Management and Social Sciences, Tribhuvan University.
# Features

User registration & login with email-based OTP verification
Profile management (bio, profile picture, location, social links)
Post creation supporting text, images, and videos
Like, comment, repost, and cross-share posts
Follow/unfollow users with a personalized feed
Search users with history tracking
In-app and email notifications for all interactions
Interaction report to view engagement with followers
Responsive UI built with Bootstrap and custom CSS

# Tech Stack
. Django · SQLite · Bootstrap · HTML/CSS/JS · Django ORM
# Development Methodology
Built following the Waterfall model across requirement analysis, design, implementation, and testing phases.
